框架说明 ​

版本要求 ​

node 版本 ​

  • node >= v20.18.0
  • npm >= 10.8.2

注意：已经通过package.json文件增加限制

常用指令 ​

安装方式 ​

shell
npm install

启动方式 ​

shell
npm run dev

打包方式 ​

shell
npm run build

代码校验 ​

shell
npm run lint:eslint
npm run lint:stylelint
npm run lint:prettier
npm run lint:spell-check

文档相关 ​

shell
npm run docs:dev
npm run docs:build
npm run docs:preview

开发工具 ​

推荐 vscode 编辑器 ​

工程默认带着.vscode文件，文件中的配置开启了保存自动修复，如果使用WebStorm等编辑器请自行配置。

.vscode文件下包含：

  • extensions.json 插件配置文件
  • global.code-snippets 代码片段文件
  • settings.json 编辑器配置

必要 vscode 插件 ​

下面插件禁止全局禁用，项目级别的已自带安装并启用：

  • vue-official
  • eslint
  • styleLint
  • prettier
  • code-spell-checker

单元测试 ​

后续增加单元测试集成。

规范要求 ​

  • Eslint 做代码质量检查
  • StyleLint 做样式检查
  • Prettier 做代码格式化
  • Code-spell-checker 中文拼写检查插件
  • Commitlint 做提交信息规范检查

框架已经集成了以上规范，请按照要求进行代码编写。如果不按照规范编写，会导致代码质量问题。同时不按照提交规范提交代码，会导致提交信息不规范，无法提交。
